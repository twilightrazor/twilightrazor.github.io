Instructions

download RPCS3 (https://rpcs3.net/)
--OR--
download EmuDeck (https://www.emudeck.com/) [recommended]

download PS3 Firmware [http://dus01.ps3.update.playstation.net/update/ps3/image/us/2024_0227_3694eb3fb8d9915c112e6ab41a60c69f/PS3UPDAT.PUP]

 

Install emudeck or rpcs3

extract LittleBigPlanet3 to the ps3 roms folder (emudeck makes one, if not, something like THICCBOI:/Emulation/Roms/PS3 would work)

extract the updates and dlc to a TEMP folder

in RPCS3, File > Install Firmware, click the downloaded PS3UPDAT.PUP, let it do its thing.

#Enable Online Play
Configuration > RPCN

File > Add Games, (wherever you placed the rom folder)

now, File > Install Package, install the firstandalone update, after you can do the rest of the updates at once.
Next install the DLC, game order doesnt matter but number does.
After the DLC reinstall the final game update (1.26)

Right-click LittleBigPlanet 3 within RPCS3 and choose Manage Game Patches
enable Beacon Custom Server
Delete the gameid at the top.
Under All titles - Warning: These patches apply to all games!
Enable Disable SPU MLAA - LittleBigPlanet 2, LittleBigPlanet 3, LittleBigPlanet Hub

Game *should now work

 

 

*the game may still not work
